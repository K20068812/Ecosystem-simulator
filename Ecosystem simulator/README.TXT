The Savannah Simulation v2021.02.24

Authors: Amman Kiani (20068812) and Aarjav Jain (20050399). 

This simulation models a predator-prey ecosystem with multiple different
species. The behaviour of the species is influnced by factors such as
weather, disease and time of day. 


To start this project, create an object of simulator class and run it's 
methods. Alternatively, run the main method included in Simulator class.
